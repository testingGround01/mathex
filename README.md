# Mathex

This project is a simple math practice dashboard. HTML is generated from modular EJS templates.

## Requirements
- Node.js >= 14

## Installation
```
npm install
```

## Building
Render the EJS templates into `index.html`:
```
npm run build
```

## Running
You can serve the page locally using `http-server`:
```
npm start
```
This builds the templates and launches a local server at <http://localhost:8080>.

The generated `index.html` works with the existing `style.css` and `script.js` files.
